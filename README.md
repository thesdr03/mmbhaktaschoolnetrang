# 🏫 Smt. M.M. Bhakta High School Website

Official website for **Smt. M.M. Bhakta High School, Netrang**

A modern, responsive school website built with **HTML5**, **Tailwind CSS**, and **JavaScript**.

---

## 📋 Features

- 📸 **Photo Galleries** - Event galleries, sports, cultural events, and more
- 👥 **Staff Directory** - Complete staff information
- 📚 **Alumni Directory** - Connect with school alumni
- 🎓 **Management Info** - School management and administration details
- 📞 **Contact Form** - Information gateway form with Formspree integration
- 📱 **Responsive Design** - Works perfectly on mobile, tablet, and desktop
- ⚡ **Fast Loading** - Optimized performance with CDN resources
- 🎨 **Modern UI** - Beautiful gradient design with smooth animations

## 🛠️ Quick Start (Local Development)

### Prerequisites
- Python 3 installed ([Download](https://www.python.org/downloads/))
- Git ([Download](https://git-scm.com/))

### Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR-USERNAME/school-website.git
   cd school-website
   ```

2. **Run the local server:**
   ```powershell
   # Windows
   run-server.bat
   
   # Or manually:
   python -m http.server 8000
   ```

3. **Open in browser:**
   ```
   http://localhost:8000
   ```

4. **Access specific pages:**
   ```
   http://localhost:8000/index%20main.html
   http://localhost:8000/staff.html
   http://localhost:8000/alumni-directory.html
   http://localhost:8000/management.html
   ```

## 🚀 Deploy on GitHub Pages

To host this website for free on GitHub Pages:

1. Ensure your repository is public
2. Go to Settings → Pages
3. Set source to "Deploy from a branch" → `main` → `root`
4. Your site will be live at: `https://YOUR-USERNAME.github.io/school-website/`

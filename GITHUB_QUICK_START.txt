🆘 TROUBLESHOOTING
===================

❌ "fatal: not a git repository"
   → You're not in the school website folder
   → Run: cd "C:\Users\Admin\OneDrive\Desktop\school website"

❌ "remote origin already exists"
   → Run: git remote remove origin
   → Then: git remote add origin https://...

❌ "Authentication failed"
   → Use Personal Access Token (not GitHub password)
   → Make sure token has "repo" scope selected

❌ "nothing to commit"
   → All files are already committed
   → Continue to Step 4️⃣ above
